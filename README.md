# Tetris
Классический тетрис<br>
Управление:<br>
ESC - Пауза<br>
A, D - влево, вправо<br>
W - Повернуть фигуру<br>
S - Ускоренный спуск фигуры<br>
*Стрелки работают также*<br>
Пробел - Постановка фигуры<br>
R - Перезапустить игру<br>

Classic tetris<br>
Controls:<br>
ESC - Pause<br>
A, D - left, right<br>
W - Rotate figure<br>
S - fast falling<br>
*You can also use arrows*<br>
SPACE - Drop figure<br>
R - Restart game<br>
